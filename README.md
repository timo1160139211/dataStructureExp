小结

起初,本人是非常抵触敲数据结构代码的,因为书上的实例是伪代码,并且仅是二叉树,
想起曾经写过二叉树的代码,遂翻出来参考.
从创建到遍历,一步一步走下去,发现之前的操作方法都可以为后来的方法做铺垫,
不管是在思想上还是代码上.
而后思入泉涌,进行比较顺利.

检讨一下,个人非常任性的取消了struct里的parent回指指针,导致后来查找修改方法写起来非常绕.虽然通过不同的思路也能实现,但是实在是绕.深刻反省.
